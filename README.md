# Ecommerce Backend API

A Django REST Framework based E-Commerce Backend API.

## Features
- JWT Authentication
- User Authentication
- Product Management
- Cart Management
- Order Management

## Technologies Used
- Python
- Django
- Django REST Framework
- SQLite

## How to Run

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
